# mini-job-board-api
A simplified backend API for a job board application built with Django REST Framework (DRF), featuring JWT authentication, custom permissions, and job filtering.
# Project Overview
This is a Django REST Framework (DRF) backend API designed for a simplified job board application. It allows users to register, obtain JWT tokens, manage their own companies, post jobs linked to their company, and apply to available job listings.Core Objectives AchievedModels: Company, Job, and JobApplication models implemented.Authentication: JWT-based authentication using djangorestframework-simplejwt.API Endpoints: Complete set of required endpoints (/api/register/, /api/token/, /api/companies/, /api/jobs/, /api/jobs/{id}/apply/).Permissions: Custom Object-Level Permissions ensure users can only manage resources they own.
